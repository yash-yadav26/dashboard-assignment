# 📊 Dashboard Assignment

This project is a responsive Employee Self Service Dashboard built using HTML, CSS, Bootstrap, jQuery, and JavaScript.

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- jQuery

---

## 📱 Features
- Fully responsive design (Mobile, Tablet, Desktop)
- Sidebar navigation
- Dashboard cards (Meetings, Tasks, Events)
- To-do list section
- Meeting schedule section
- Feed/Activity panel
- Progress bars for projects
- Task review section
